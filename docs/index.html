This lab aims to introduce the Renesas RL78/G23 indoor air quality demonstration board and in particular highlight it's capability Publish the gathered data online over a hosted website using DA16200.
Covering also the following:

  1. Creating & configuraing a project with RL78/G23
  2. Usage of key peripherals including UART, I2C& Timers
  3. "Free" snippets of useful code such as DA16200 connectivity & Hs4001 library
  4. Building & debugging the project

Please read the important notes below carefully and then click the Get Started! button to start the lab.
